# 📊 Python for CFOs — The Complete 7-Notebook Course
### 📺 By Christian Martinez: AI for Finance
👉 Subscribe: https://www.youtube.com/@christianmartinezAIforFinance

A hands-on, beginner-to-advanced Python course built for finance leaders. Every notebook runs in **Google Colab** with zero setup — data is embedded, with optional downloadable CSVs for realism.

## 🚀 How to start (60 seconds)
1. Go to https://colab.research.google.com
2. File → Upload notebook → pick any `.ipynb` below
3. (Optional) Upload the matching CSV from `/datasets` — or just run; each notebook auto-generates its data if no file is found.
4. Press **Shift+Enter** on each cell.

## 📁 Datasets (in /datasets)
- `monthly_pl.csv` — 36 months of P&L
- `customers.csv` — 500 customers (churn, LTV inputs)
- `daily_sales.csv` — 2 years daily sales (forecasting)
- `capex_projects.csv` — 8 investment projects
- `ar_aging.csv` — 200 invoices
- `asset_returns.csv` — 4-asset daily returns (risk/VaR)
